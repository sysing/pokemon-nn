pokemon-nn — Project State

github.com/sysing/pokemon-nn  ·  Toy neural network for classifying Pokemon card types from card images. Trained on M4 MacBook Air (MPS). 5 releases over 2 days.

Architecture

Input

[batch, 4, 224, 224]

Channel 0: Red
Channel 1: Green
Channel 2: Blue
Channel 3: Luminance
 (0.299R + 0.587G + 0.114B)

200,704 float values per image. The 4th channel is the key innovation from v0.3+.

Model

Conv2d(4→32) → BN → ReLU → MaxPool
Conv2d(32→64) → BN → ReLU → MaxPool
Conv2d(64→128) → BN → ReLU → MaxPool
Conv2d(128→256) → BN → ReLU → AvgPool
Dropout(0.5) → Linear(256→10)

~600K parameters. 10 output classes.

Training Config

OptimizerAdamW, lr=1e-3, wd=1e-4
ScheduleReduceLROnPlateau (factor=0.5, patience=3)
LossCrossEntropy (class-weighted, label_smoothing=0.1)
Batch32
AugmentFlip, Rotate, ColorJitter
Seed42 (deterministic)

Speed Optimizations

AMPMixed precision (float16 on MPS)
PreloadImages cached in RAM (~750MB)
Workers2 DataLoader workers
Epoch time~18-25s on M4
30 epochs~12-15 minutes total

Key Findings

1. 4-Channel Input Is the Breakthrough

Stacking grayscale as a 4th channel ([R,G,B,Gray]) lets the model learn a soft gating mechanism: attend to RGB for easy color-based types, fall back to grayscale layout/texture features for ambiguous pairs.

v0.3 (4-channel) is the highest overall accuracy at 94.7%, with Metal +45pp over baseline.

2. Curriculum Pretrain Helps the Hardest Class

Phase 1: train on Colorless+Metal only. Phase 2: add all 10 classes. Metal hits 95.2% — best ever. But it shifts errors rather than reducing total errors (Colorless and Lightning regressed).

3. Sample Size Matters

Dragon had 100% accuracy in v0.1–v0.3 with only 5 validation samples. Expanding to 19 val samples revealed true accuracy of ~89%. Always check sample counts alongside accuracy metrics.

4. Metal vs Colorless Is a Genuinely Hard Problem

Both types share silver/grey color palettes. Even with our best techniques (4-channel + curriculum), Colorless peaks at 94% and Metal at 95%. The model needs layout/textual features that a 600K CNN can barely extract.

CLI Reference

Train

python src/train.py --run v0.6 --epochs 30 --preload --workers 2

With curriculum pretrain:

python src/train.py --run v0.6 --epochs 30 \
  --pretrain Colorless Metal --pretrain-epochs 5

Predict

python src/predict.py path/to/card.jpg
python src/predict.py path/to/card.jpg \
  --model runs/v0.5/model.pt --top-k 5

Visualize (live)

python src/viz.py runs/v0.6/metrics.csv

Post-hoc Plot

python src/plot.py
